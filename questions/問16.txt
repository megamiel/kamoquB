次のプログラムと記述中の「　a　」と「　b　」に入れる正しい答えを、解答群の中から選べ。
　　関数isCollatz はコラッツ予想が1～指定された上限値まで全ての整数に適用されているかを論理型で返す関数である。
　　コラッツ予想とは、どんな値であっても、以下の操作を有限回行ううちに必ず1に到達するという予想である。
　　操作内容
　　・初期値をnとする
　　・nが偶数の場合、nを2で割る
　　・nが奇数の場合、nに3かけて1を足す

〔プログラム〕
　　〇論理型:isCollatz(整数型:maxLim)
　　　for(iを1からmaxLimまで1ずつ増やす)
　　　　if(modular(i)が1と等しくない)
　　　　　return false
　　　　endif
　　　endfor
　　　return true
    
　　〇整数型:modular(整数型:n)
　　　while(nが1と等しくない)
　　　　if(「　a　」)
　　　　　n ← n / 2
　　　　else
　　　　　n ← n * 3 + 1
　　　　endif
　　　endwhile
　　　return n


　　上記のプログラムは、コラッツ予想を肯定的に捉えて作成された関数であり、コラッツ予想が正しくなかった際、関数modularの「　b　」


　解答群
　　ア　a:n mod 2が0と等しい
　　　　b:戻り値はfalseとなる。

　　イ　a:n mod 2が1と等しい
　　　　b:戻り値はfalseとなる。

　　ウ　a:n mod 2が0と等しい
　　　　b:繰り返し処理は無限回繰り返される。

　　エ　a:n mod 2が1と等しい
　　　　b:繰り返し処理は無限回繰り返される。
